# QR Generator with Customization
 
## Описание
Современный и стильный сервис для генерации кастомизированных QR-кодов уровня профессионального продукта. Приложение поддерживает глубокую настройку внешнего вида (стиль точек, углов, цвета, градиенты, загрузку логотипов) и разные типы данных (Ссылки, Текст, Wi-Fi и Визитки vCard).  
Проект построен по полностековой (Fullstack) архитектуре и включает как клиентское приложение, так и бэкенд с сохранением истории сгенерированных кодов.

## Стек технологий
- **Frontend:** React 18, Vite, Tailwind CSS v4
- **Backend:** Node.js, Express
- **База данных:** JSON Storage (db.json)
- **Основные библиотеки:** `qr-code-styling` (отрисовка Canvas), `concurrently` (одновременный старт)

## Инструкция по запуску

Для удобства проект разбит на рабочие области (Workspaces). Все зависимости для бэкенда и фронтенда устанавливаются автоматически из корневой директории.

**Шаг 1: Установка зависимостей**
Откройте терминал в корневой папке проекта и выполните команду:
```bash
npm install
```
*(Эта команда установит как корневые зависимости, так и пакеты для frontend и backend благодаря npm workspaces)*

**Шаг 2: Запуск приложения (Development mode)**
После успешной установки выполните команду:
```bash
npm run dev
```
*(Эта команда через `concurrently` одновременно запустит сервер Node.js на порту 3001 и скомпилирует Frontend на порту 5173)*

Теперь вы можете перейти в браузере по адресу `http://localhost:5173/` и начать использовать Генератор QR-кодов.

## Структура проекта
- `/frontend` — исходный код клиентского React-приложения. Содержит интерфейс, логику работы с локальной историей, управление темами и стейтами превью Canvas.
- `/backend` — исходный код Express-сервера. Содержит API-эндпоинты (REST) для создания (`POST /api/qr/generate`), получения (`GET /api/qr/history`) и удаления (`DELETE /api/qr/clear`, `DELETE /api/qr/:id`) записей в локальной базе данных `db.json`. 
- `/docs` — документация проекта и внутренние рефлексивные материалы о процессе разработки.
