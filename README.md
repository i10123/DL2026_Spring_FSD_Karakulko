# QR Studio 🚀

Современный и стильный сервис для генерации кастомизированных QR-кодов уровня профессионального продукта. 
Поддерживает глубокую настройку внешнего вида (стиль точек, углов, прозрачность, цвета, загрузку логотипов) и разные типы данных (Ссылки, Текст, Wi-Fi, Электронная почта и Визитки vCard).

## ✨ Быстрый старт

Запуск проекта осуществляется одной командой (используя `concurrently`). Убедитесь, что у вас установлен Node.js v18+.

```bash
# 1. Склонируйте репозиторий и перейдите в папку
git clone <repository-url>
cd DL2026_Spring_FSD_Karakulko

# 2. Установите зависимости (сработает сразу для корня, frontend и backend)
npm install

# 3. Запустите проект
npm run dev
```

Откройте в браузере: **http://localhost:5173**

## 🛠 Feature List (Возможности)

- [x] **Форматы данных:** URL, обычный текст, Wi-Fi сети (с шифрованием), vCard визитки, шаблоны Email и SMS.
- [x] **Продвинутый Дизайн QR:** 7 стилей точек, 4 типа рамок, кастомные углы.
- [x] **Цветовые настройки:** Поддержка градиентов (linear/radial) и сплошных заливок для фона и точек.
- [x] **Брендирование:** Загрузка собственного логотипа в центр QR-кода с очисткой фона.
- [x] **Экспорт изображений:** Форматы PNG, SVG, JPEG, WEBP.
- [x] **Масштабирование:** Качество до 4K (2400px) без потери четкости.
- [x] **История генераций:** Сохранение в базу данных и восстановление настроек в 1 клик.
- [x] **Быстрые действия:** Горячие клавиши (Ctrl+S, Ctrl+D) и мгновенное копирование картинки (Ctrl+C).

## 🗺 Roadmap (Планы развития)

1. **Пакетная генерация:** Загрузка CSV файла для автоматической генерации сотен QR-кодов по шаблону.
2. **Анимации:** Возможность скачивания GIF-анимированных QR-кодов.
3. **Облачная синхронизация:** Замена локального `db.json` на MongoDB или Supabase для хранения истории между устройствами.
4. **Шаблоны (Presets) сообщества:** Возможность делиться своими стилями QR-кодов по ссылке.

## 🧱 Структура проекта

Архитектура Fullstack Monorepo:
- `frontend/` — React 18 (Vite), Tailwind CSS v4, Framer Motion, QR-Code-Styling.
- `backend/` — Node.js, Express, Zod (Validation), JSON File DB.
- `docs/` — [Системный дизайн проекта (design.md)](docs/design.md) и AI-рефлексия.
